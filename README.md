# CPR-Migration-Graphs
Graphs for the domestic migration spread for CPR print edition

Domestic migration data from: https://www.census.gov/data/tables/time-series/demo/geographic-mobility/state-to-state-migration.html
State shapefile from: https://www.census.gov/geographies/mapping-files/time-series/geo/carto-boundary-file.html
    Speciaically the cb_2018_us_state_500k.zip file

Font for graphs from: https://fonts.google.com/selection?preview.script=Latn
